Hospital Management System
A robust Java-based console application designed to streamline hospital operations by managing patient records, doctor information, and appointment scheduling. The system utilizes JDBC for persistent data storage in a MySQL database.

##🚀 Features

    Patient Management: Add new patients with details including name, age, gender, and address.
    Doctor Records: View a comprehensive list of registered doctors and their specializations.
    Appointment Booking: Schedule appointments by checking the real-time availability of specific doctors on requested dates.
    Data Validation: Integrated checks to ensure doctors and patients exist before finalizing appointments.


##🛠️ Technology Stack

    Language: Java
    Database: MySQL
    API: JDBC (Java Database Connectivity)


##🖥️ Usage

  Run the HospitalManagementSystem.java file to start the application. You will be presented with a menu:
  Add Patient: Register a new patient.
  View Patient: Display all registered patients.
  View Doctors: Display all available doctors.
  Book Appointment: Check availability and schedule a visit.
  Exit: Close the application.
